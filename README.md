# C07 - Parametric Polymorphism in C++, Java, and C#: Generics vs. Templates and Metaprogramming Power

This repository contains three PDF documents:

- **`report.pdf`** — The main technical report, comparing how C++, Java, and C# implement parametric polymorphism (templates vs. generics) and metaprogramming, covering instantiation time, runtime representation, performance, type safety, and reflection.
- **`prompt.pdf`** — The full log of the AI prompts used to generate the report's content, including the master prompts and the individual prompts issued for each section.
- **`verification.pdf`** — The verification log documenting the review process, listing every error or imprecision found in the AI-generated content along with its correction and supporting sources.
